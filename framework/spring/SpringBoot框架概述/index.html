简单诠释Spring Boot的诞生背景，并提供了快速开发示例。

Spring Boot Version：2.7.9

1. Spring Boot的诞生背景

在没有SpringBoot之前，使用Spring框架与其它技术整合时需要经过非常繁琐的配置。

  1. 使用Spring Web框架时，需要配置web.xml。

  2. 使用Mybatis框架时，需要手动配置SessionFactory，Datasource，Connection，扫描映射接口，映射文件等。

  3. 使用Redis技术时，需要相关Redis配置。

针对以上问题，Spring官方提供了Spring Boot框架。

Spring Boot其定位就是快速搭建Spring引用。提供了高度可定制化的配置，许多使用的功能。

例如

  • 依赖版本管理（spring boot官方维护了一套常用的依赖版本控制，极大程度上解决了开发者自行配置依赖可能导致依赖冲突的情况

  • starter 快速整合依赖（官方或第三方提供的快速整合依赖

  • 自动配置（自动加载类路径上存在的配置类并进行配置

  • 应用配置文件（比如：application.yml

  • 应用构建（spring boot提供了应用打包工具
