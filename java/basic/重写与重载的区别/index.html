重写与重载的区别

March 01, 2023 363

1. 定义

重载：方法名相同，参数不同（参数个数，参数顺序，参数类型），与返回值无关。

重写：子类重写父类的方法，方法名必须相同，参数必须相同，返回值小于等于父类，异常小于等于父类。

Example：重载

Example：重写

2. 范围

重载：发生在父子类中（子类可以重载父类方法）

重写：子类只有继承父类才可以重写

3. 多态

重载：重载是编译时的多态

重写：重写是运行时的多态（JVM运行时）

4. 返回

重载：重载对返回类型没有要求

重写：重写的方法必须兼容返回值类型

5. 参数

重载：重载的方法参数可以不同

重写：重写的方法参数必须相同

6. 修饰

重载：重载对修饰符没有要求

重写：重写的方法修饰符必须大于等于被重写方法的修饰符

7. 异常

重载：重载对异常没有要求

重写：重写的方法异常必须小于等于被重写方法的异常
